# Lexical Analyzer

A web application to break the program into token. It was created as a open ended project.

Live: [https://spring-lexical-analyzer.herokuapp.com/](https://spring-lexical-analyzer.herokuapp.com/)

### Tech stack:

- Java
- Spring Boot
- Thymeleaf
- Tailwind CSS
